## AdaBoost
- 앙상블 학습의 접근법 중 연속화 방식으로 개별 학습기간 강한 의존성을 가지지만 순차적으로 생성된다.
- 약한 학습기를 강한 학습기로 향상할 수 있는 알고리즘이다.
- 편향-분산 분해 관점에서 편향을 줄이는 것에 초점을 두기 때문에 일반화 성능이 비교적 낮은 학습기들을 조합하여 견고한 학습기를 만들 수 있다.
- AdaBoost는 기초 학습기의 선형가중조합을 통해 생성된다.
